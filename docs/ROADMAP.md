# Roadmap

## MVP

- local fixture-driven analysis
- live proxy inspection over JSON-RPC
- storage layout risk detection
- authority downgrade detection
- dangerous implementation signal detection
- Markdown and JSON reports

## Strong differentiators

- proxy-address mode that reads current on-chain control context
- authority graph diffing across Safe, timelock, ProxyAdmin, and implementation guards
- risk-triggered fork simulation
- upgrade review policy files per protocol
- GitHub Action that comments on pull requests with a verdict

## Resume-friendly milestones

1. `v0.1`: local approval engine with sample corpora
2. `v0.2`: on-chain proxy introspection
3. `v0.3`: governance-path downgrade analysis
4. `v0.4`: risk-triggered dynamic testing hooks
5. `v0.5`: CI and PR review integration
